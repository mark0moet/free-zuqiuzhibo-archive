# free-zuqiuzhibo-archive

## 项目简介

本项目是一个用于归档和发布多个独立 HTML 页面的仓库。这些页面均为静态网页，不依赖于任何特定后端服务，可直接在浏览器中打开查看。仓库内容保持原始状态，仅做收集与存档用途。

## 目录结构

```
.
├── index.html          # 主页/导航页
├── pages/              # 存放所有 HTML 页面文件
│   ├── page1.html
│   ├── page2.html
│   └── ...
└── README.md           # 本文件
```

- `index.html`：提供页面列表及基本导航。
- `pages/`：所有归档的 HTML 文件均存放于此目录下，每个文件可独立访问。

## 页面归档说明

- 归档内容均为 HTML 格式的静态页面，不包含可执行脚本或外部资源（除 CDN 外）。
- 页面内容可能包含文本、图片、样式等，均来源于公开可访问的素材。
- 本仓库不存储任何用户数据、登录信息或动态内容。
- 归档仅用于信息保存与参考，不保证页面功能或交互的完整性。
